The Pandas Library¶

The pandas library contains useful functions for reading and manipulating data. It organizes the CSV file into a data frame. In this notebook we will use pandas to read in our data and explore it.

Palmer Penguins Data¶

We will use the penguins_size dataset, compiled by Alison Horst

This is data about three types of penguins that were studied in Antarctica.

The data we are using is available at Parul Pandey's Kaggle page

Select the penguins_size.csv file and click to download, and save it in the same folder as this notebook so that you can read it in easily.

Using Pandas¶

To use the pandas library, we have to import it. We usually use pd as the alias to refer to it as a shortcut.

Groupings using groupby¶

When we have many categories of data, we would like to compare values between the categories The groupby() function can be used group the data by a specific column, for example if we want to find the means for the numeric values for each species:

In [33]:
# find the mean for each group of species
df.groupby("species").mean()
Out[33]:
culmen_length_mm culmen_depth_mm flipper_length_mm body_mass_g

Chinstrap 48.833824 18.420588 195.823529 3733.088235

This calculates the mean of each of the numeric values, for each group of species. You can also count the number of values recorded

print(df.groupby(['species']).nunique())
